# Operational Data Platform for Fleet & Issue Management

Built an end-to-end operational data platform in Palantir Foundry to manage fleet performance and issue tracking for a car leasing business.

## Overview
This project simulates a real-world business scenario where a company needed a centralized system to monitor vehicles, track maintenance history, and manage reported issues.

The solution integrates multiple datasets, applies data transformations, and delivers an interactive application for operational decision-making.

## Key Features
- Integrated multiple datasets (cars, models, maintenance, issues)
- Cleaned and standardized inconsistent data (dates, formatting, invalid records)
- Built data pipelines for joins, aggregations, and transformations
- Designed ontology for cars and issues with relationships
- Developed interactive application for:
  - Fleet dashboard
  - Issue tracking and prioritization
  - Real-time updates

## Architecture
Raw Data → Data Transformation → Ontology Modeling → Interactive Application

## Tech Stack
Palantir Foundry, Pipeline Builder, Ontology Manager, Workshop
